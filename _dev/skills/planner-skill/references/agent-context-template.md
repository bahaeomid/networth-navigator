# Agent Context File Template

Use this template in Stage 3 to generate the initial `CLAUDE.md`, `AGENTS.md`,
`.cursor/rules/base.mdc`, or equivalent file for the project.

## Two modes of use

**New project (no existing file):** Generate the full scaffold below.

**Existing file:** Append only a `## Project Plan — [date]` section with the
planning-specific content. Never overwrite an existing agent context file.

---

## File naming by environment

| Agent environment | File name | Location |
|------------------|-----------|----------|
| Claude Code | `CLAUDE.md` | Project root |
| Codex, OpenCode | `AGENTS.md` | Project root |
| Cursor | `.cursor/rules/base.mdc` | Project root (always = ruleset) |
| Unknown / portable | `AGENTS.md` | Project root |

For multi-tool projects: create `AGENTS.md` and symlink `CLAUDE.md -> AGENTS.md`.
If unsure, use `AGENTS.md` — it is the most portable choice.

---

## What this file is NOT

- Not a feature list or roadmap
- Not an implementation guide or tutorial
- Not a place for verbose explanations
- Not static — it will grow as development reveals operational rules

The agent reads this file at the start of every session. Every line competes for
its attention. Keep it dense, specific, and actionable.
